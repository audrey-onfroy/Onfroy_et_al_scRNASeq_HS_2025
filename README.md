### The molecular signature of hair follicles enables the classification of Hidradenitis suppurativa patients into inflammatory, keratinizing, and mixed endotypes

This repository contains the code to ensure the traceability of the analysis of single-cell RNA Sequencing data from the raw count matrices to the figures in the associated publication. Please open [index.html](https://audrey-onfroy.github.io/scrnaseq_hair_follicles_hidradenitis_suppurativa/) file in any browser, and navigate through all files in this repository.
